# Chronomètre Web

Un chronomètre simple réalisé en **HTML, CSS et JavaScript**, fonctionnant directement dans le navigateur, sans dépendance externe.

## Fonctionnalités
- Démarrer le chronomètre
- Mettre en pause
- Réinitialiser le temps
- Affichage heures : minutes : secondes

## Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)

## Utilisation
1. Télécharger ou cloner le projet
2. Ouvrir le fichier `index.html` dans un navigateur
3. Cliquer sur **Démarrer**
